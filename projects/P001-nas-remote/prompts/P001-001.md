# P001-001 - 系統設計

## 任務

設計一個聯絡人管理系統（網頁版）的架構。

## 需求

| 項目 | 內容 |
|------|------|
| **類型** | 網頁應用 |
| **功能** | 新增、編輯、刪除、搜尋 |
| **技術** | Python Flask（後端）+ 網頁前端 |
| **資料儲存** | JSON 檔案（不需資料庫） |

## 資料模型

### 聯絡人欄位

| 欄位 | 類型 | 說明 |
|------|------|------|
| id | string | 唯一識別碼（UUID） |
| name | string | 姓名（必填） |
| phone | string | 電話 |
| email | string | 電子郵件 |
| company | string | 公司 |
| notes | string | 備註 |
| created_at | string | 創建時間 |
| updated_at | string | 更新時間 |

## 產出要求

1. **系統架構圖** - 說明前端、後端的互動流程
2. **API 規格** - REST API 端點設計
   - GET /contacts - 取得所有聯絡人
   - GET /contacts/<id> - 取得單一聯絡人
   - POST /contacts - 新增聯絡人
   - PUT /contacts/<id> - 更新聯絡人
   - DELETE /contacts/<id> - 刪除聯絡人
   - GET /contacts/search?q=<keyword> - 搜尋聯絡人
3. **資料儲存** - contacts.json 檔案結構
4. **前端頁面規劃** - 需要哪些頁面/視圖

## 輸出格式

請輸出完整的系統設計文件，包含所有規格說明。
