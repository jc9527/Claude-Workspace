## 1. 系統概觀

### 1.1 目的

重建一個跨平台的檔案管理系統，可在 Ubuntu（Linux）上運行，使用 .NET 8 + Blazor 技術。

### 1.2 來源系統

| 項目 | 現況 |
|------|------|
| 原始專案 | devpro-tw/FileManager |
| Framework | .NET Framework 4.7 |
| UI 框架 | DevExpress MVC v18.2（Windows only）|
| 儲存位置 | `C:\D\temp`（Windows 路徑）|

## 5. DevExpress 替代方案（重要）

### 5.1 替代元件庫比較

| 元件庫 | FileManager | .NET 8 支援 | 免費程度 | 成熟度 |
|--------|-------------|-------------|----------|--------|
| **Radzen Blazor** | ✅ 有 | ✅ 完全支援 | 免費（MIT）| 高 ⭐ |
| **AntDesign Blazor** | ❌ 無（需自定義）| ✅ | 免費（MIT）| 高 |
| **MudBlazor** | ❌ 無（需自定義）| ✅ | 免費（MIT）| 高 |
| **Telerik UI for Blazor** | ✅ 有 | ✅ | **付費** | 最高 |
| **DevExpress Blazor** | ✅ 有 | ✅ | **付費** | 最高 |

### 5.2 各替代方案詳細說明

#### 🥇 Radzen Blazor（推薦）
```
優點：
- 完全免費（MIT License）
- 有現成的 FileManager 元件
- 完全支援 .NET 8
- 開放原始碼

缺點：
- FileManager 功能相對基礎
- 文件較少

安裝：
dotnet add package Radzen.Blazor
```

#### 🥈 Telerik UI for Blazor
```
優點：
- 有完整的 FileManager 元件
- 技術支援
- 非常成熟

缺點：
- **付費**（商業授權）
- 價格較高

官網：https://www.telerik.com/blazor-ui
```

#### 🥉 AntDesign Blazor
```
優點：
- UI 非常美觀
- 大量元件
- 完全免費

缺點：
- **沒有 FileManager 元件**
- 需要自己组合 Tree + Table + Upload 元件

安裝：
dotnet add package AntDesign.Blazor
```

#### 4. MudBlazor
```
優點：
- Material Design 風格
- 完全免費

缺點：
- **沒有 FileManager 元件**
- 需要自己组合元件

安裝：
dotnet add package MudBlazor
```

### 5.3 自定義實作方案（最後手段）

如果不想用付費元件，又覺得 Radzen 功能不足：

| 組合 | 功能 |
|------|------|
| **AntDesign Tree** | 資料夾樹狀結構 |
| **AntDesign Table** | 檔案列表（含排序、篩選）|
| **AntDesign Upload** | 檔案上傳 |
| **AntDesign Modal** | 重新命名、刪除確認 |
| **ASP.NET Core Static Files** | 檔案下載 |

### 5.4 DevExpress 具體功能對應

| DevExpress 功能 | Radzen Blazor | Telerik | 自定義 |
|-----------------|---------------|---------|--------|
| 資料夾樹 | ✅ Tree | ✅ Tree | AntDesign Tree |
| 檔案列表 | ✅ DataGrid | ✅ Grid | AntDesign Table |
| 拖拉上傳 | ✅ Upload | ✅ Upload | AntDesign Upload |
| 麵包屑 | ❌ 需自定義 | ✅ Breadcrumb | 需自定義 |
| 多選下載 | ❌ 需自定義 | ✅ | 需自定義 |
| 縮圖模式 | ❌ 需自定義 | ✅ Thumbnail | 需自定義 |

## 6. 建議的 UI 框架選擇

| 優先順序 | 框架 | 理由 |

## 10. 風險與對策

| 風險 | 等級 | 對策 |
|------|------|------|
| Blazor 元件庫不支援檔案管理 | 🟡 中 | 使用多個元件組合或自定義 |
