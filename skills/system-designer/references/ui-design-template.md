# UI 設計模板

---

## 頁面設計基本格式

```markdown
# [頁面名稱]

**路由**：`/path/to/page`
**所屬 Application**：[Application 名稱]
**類型**：[Web Form / WinForm / Console]
**說明**：[頁面功能描述]

---

## 頁面佈局

```
┌─────────────────────────────────────────────────────────────┐
│  Header / 頁首                                             │
├─────────────────────────────────────────────────────────────┤
│  Navigation / 導航列                                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  主要內容區域                                               │
│  ┌─────────────────────┬─────────────────────────────┐    │
│  │                     │                             │    │
│  │   Left Panel       │   Right Panel               │    │
│  │   (篩選/列表)       │   (詳情/編輯)               │    │
│  │                     │                             │    │
│  └─────────────────────┴─────────────────────────────┘    │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  Footer / 頁尾                                             │
└─────────────────────────────────────────────────────────────┘
```

## 欄位定義

### 輸入欄位

| # | 欄位名稱 | 欄位 ID | 類型 | 必填 | 預設值 | 驗證規則 | 說明 |
|---|----------|---------|------|------|--------|----------|------|
| 1 | 會員名稱 | txtName | TextBox | 是 | - | 最少 2 個字元 | |
| 2 | Email | txtEmail | TextBox | 是 | - | 格式驗證 | |
| 3 | 電話 | txtPhone | TextBox | 否 | - | 格式驗證 | |
| 4 | 狀態 | cboStatus | ComboBox | 是 | 啟用 | - | |

### 顯示欄位

| # | 欄位名稱 | 欄位 ID | 類型 | 說明 |
|---|----------|---------|------|------|
| 1 | 建立日期 | lblCreatedAt | Label | 格式：yyyy-MM-dd HH:mm |
| 2 | 更新日期 | lblUpdatedAt | Label | 格式：yyyy-MM-dd HH:mm |

## 按鍵功能

| # | 按鍵名稱 | 按鍵 ID | 圖示 | 權限 | 動作 | 說明 |
|---|----------|---------|------|------|------|------|
| 1 | 新增 | btnAdd | + | Admin | 開啟新增表單 | |
| 2 | 儲存 | btnSave | 💾 | Admin | 驗證並儲存資料 | |
| 3 | 取消 | btnCancel | ✖ | Admin | 清除表單/關閉視窗 | |
| 4 | 刪除 | btnDelete | 🗑 | Admin | 確認後刪除 | 需二次確認 |
| 5 | 查詢 | btnSearch | 🔍 | All | 執行查詢 | |
| 6 | 重設 | btnReset | ↺ | All | 重設篩選條件 | |

| 權限 | 說明 |
|------|------|
| All | 所有使用者 |
| Admin | 系統管理員 |
| User | 一般使用者 |

## 狀態說明

### 資料狀態

| 狀態 | 說明 | 顯示方式 |
|------|------|----------|
| Normal | 正常 | 正常顯示 |
| Loading | 載入中 | Spinner / Loading 圖示 |
| Error | 錯誤 | 紅色提示訊息 |
| Empty | 無資料 | 「查無資料」提示 |

### 表單狀態

| 狀態 | 說明 |
|------|------|
| Create | 新增模式 |
| Edit | 編輯模式 |
| View | 唯讀模式 |
| Delete | 刪除確認 |

---

## 驗證規則

| 欄位 | 規則 | 錯誤訊息 |
|------|------|----------|
| Name | Required, MinLength(2) | 「名稱至少需要 2 個字元」 |
| Email | Required, EmailFormat | 「Email 格式不正確」 |
| Phone | Optional, PhoneFormat | 「電話格式不正確」 |
| Status | Required | 「狀態為必填」 |

---

## 事件順序

```
1. 頁面載入
   └── 顯示 Spinner
       └── 抓取資料
           └── 顯示資料 / 顯示無資料

2. 使用者點擊「查詢」
   └── 顯示 Spinner
       └── 執行查詢
           └── 更新列表

3. 使用者點擊「新增」
   └── 清除表單
       └── 設定為 Create 模式
           └── 開啟表單

4. 使用者點擊「儲存」
   └── 執行驗證
       ├── 失敗 → 顯示錯誤訊息
       └── 成功 → 儲存資料
                   └── 關閉表單 / 更新列表
```

## 範例：會員列表頁面

```markdown
# 會員列表頁面

**路由**：`/members`
**所屬 Application**：會員管理系統
**類型**：Web Form (Blazor)
**說明**：顯示所有會員列表，支援篩選和分頁

---

## 頁面佈局

```
┌─────────────────────────────────────────────────────────────┐
│  會員管理                                                    │
├─────────────────────────────────────────────────────────────┤
│  [搜尋框] [狀態下拉] [查詢] [重設]           [+ 新增會員]    │
├─────────────────────────────────────────────────────────────┤
│  ┌─────┬──────────┬─────────┬────────┬─────────┬────────┐   │
│  │ 勾選 │ 會員名稱  │  Email  │  電話  │  狀態   │ 操作   │   │
│  ├─────┼──────────┼─────────┼────────┼─────────┼────────┤   │
│  │  □  │ 張三     │ z@ex... │ 0912.. │ 啟用    │ 👁 📝 🗑│   │
│  │  □  │ 李四     │ li@ex... │ 0933.. │ 停用    │ 👁 📝 🗑│   │
│  └─────┴──────────┴─────────┴────────┴─────────┴────────┘   │
├─────────────────────────────────────────────────────────────┤
│  第 1/10 頁，共 100 筆    [<] [1] [2] ... [10] [>]        │
└─────────────────────────────────────────────────────────────┘
```

---

## 欄位定義

### 列表欄位

| # | 欄位名稱 | 欄位 ID | 類型 | 寬度 | 排序 |
|---|----------|---------|------|------|------|
| 1 | 勾選 | colSelect | CheckBox | 30px | - |
| 2 | 會員名稱 | colName | TextLink | 150px | 可排序 |
| 3 | Email | colEmail | Text | 200px | 可排序 |
| 4 | 電話 | colPhone | Text | 120px | - |
| 5 | 狀態 | colStatus | Badge | 80px | - |
| 6 | 操作 | colActions | ButtonGroup | 120px | - |

### 篩選區域

| # | 欄位名稱 | 欄位 ID | 類型 | 預設值 |
|---|----------|---------|------|--------|
| 1 | 搜尋關鍵字 | txtSearch | TextBox | - |
| 2 | 狀態 | cboStatus | DropDownList | 全部 |
