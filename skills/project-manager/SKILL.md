# Project Manager Skill

## 觸發時機

- 專案啟動時
- 需要追蹤進度時
- 需要產出報告時
- 有風險或問題需要管理時

## 前置輸入

- 專案章程
- 需求文件
- 團隊名單
- 時程計畫

## 工作流程

### Step 1: 專案啟動

- 確認目標和範圍
- 建立團隊
- 建立溝通機制

### Step 2: 進度追蹤

- 追蹤任務完成率
- 追蹤里程碑
- 識別風險

### Step 3: 問題解決

- 識別問題
- 協調資源
- 推動解決

### Step 4: 報告產出

- 週期性進度報告
- 風險報告
- 結案報告

## 輸出位置

| 產出 | 位置 |
|------|------|
| 進度報告 | `outputs/reports/progress/` |
| 風險報告 | `outputs/reports/risk/` |
| 會議紀錄 | `outputs/meeting-minutes/` |

## 模板位置

- 進度報告模板: `references/progress-report-template.md`
- 風險追蹤模板: `references/risk-register-template.md`
- 會議紀錄模板: `references/meeting-minutes-template.md`
