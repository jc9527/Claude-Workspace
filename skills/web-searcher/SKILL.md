# Web Searcher Skill

## 觸發時機

- 需要對外搜尋資訊時
- 定期抓取新聞時
- 競爭分析時
- 技術研究時

## 工作流程

### Step 1: 理解搜尋目標

- 明確搜尋主題
- 選擇適當關鍵字
- 決定搜尋範圍

### Step 2: 執行搜尋

- 使用 Web Search 工具
- 設定搜尋數量（通常 3-5 條）
- 加入時間過濾（如需要）

### Step 3: 分析結果

- 讀取摘要
- 評估相關性
- 過濾不相關內容

### Step 4: 彙整產出

- 格式化為 Markdown
- 標註來源
- 給出簡短分析

## 常用搜尋模式

### AI 新聞
- `OpenAI ChatGPT GPT artificial intelligence`
- `Anthropic Claude AI model`
- `Google Gemini AI Bard`
- `Microsoft Copilot AI`
- `Meta AI Llama`

### 技術趨勢
- `artificial intelligence trends 2026`
- `machine learning breakthroughs`
- `AI regulation policy`

### 競爭情報
- `company_name AI news`
- `company_name product launch`

## 輸出格式

```markdown
## 搜尋主題：[主題]

### 結果摘要
- [標題](URL) — 簡短描述
- ...

### 分析
- 重點發現
- 趨勢觀察
```
